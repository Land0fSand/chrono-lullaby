# ============================================================
# 频道组开关功能使用示例
# ============================================================

示例 1: 临时禁用某个频道组
--------------------------------------
在 config.yaml 中找到要禁用的频道组，将 enabled 改为 false：

  - name: "唏噓電臺Beta1"
    enabled: false  # 改为 false 即禁用此频道组
    telegram_chat_id: "-1002794044093"
    ...

保存后，下一轮下载会自动跳过这个频道组，无需重启！


示例 2: 重新启用频道组
--------------------------------------
将 enabled 改回 true：

  - name: "唏噓電臺Beta1"
    enabled: true  # 改为 true 即重新启用
    telegram_chat_id: "-1002794044093"
    ...

保存后，下一轮下载会自动包含这个频道组。
